RESUMO do que eu aprendi (18/3)

git clone -> baixa o repositório do GitHub
git status -> mostra o estado dos arquivos
git diff -> mostra o que foi alterado
git add -> escolhe os arquivos que quero salvar.
git commit -m "mensagem" -> salva o que foi escolhido no add 
git push -> envia para o GitHub

ATALHOS QUE APARECERAM NA AULA

Ctrl + Alt + T
Abre o terminal no Linux.

Ctrl + Shift + V
Cola texto no terminal.

Ctrl + H
Normalmente mostra ou oculta arquivos ocultos, dependendo do programa.
